
# What is LUPY?
LUPY is a new way to run Lua using Python!

Here's a simple example:
```lua
Name = input("Enter your name: ")
print(Name)
print(fill("Your name is: ", Name, "!"))
-- The 'fill' function simply concatenates everything in order
-- output will be -> Your name is: NR!
```
